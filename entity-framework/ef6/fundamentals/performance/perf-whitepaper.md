### <a name="71-inheritance-strategies"></a><span data-ttu-id="57059-818">7,1 стратегии наследования</span><span class="sxs-lookup"><span data-stu-id="57059-818">7.1       Inheritance Strategies</span></span>

<span data-ttu-id="57059-819">При использовании Entity Framework в качестве стратегии наследования следует учитывать еще одно снижение производительности.</span><span class="sxs-lookup"><span data-stu-id="57059-819">Another performance consideration when using Entity Framework is the inheritance strategy you use.</span></span> <span data-ttu-id="57059-820">Entity Framework поддерживает три базовых типа наследования и их сочетания:</span><span class="sxs-lookup"><span data-stu-id="57059-820">Entity Framework supports 3 basic types of inheritance and their combinations:</span></span>

Каждый набор наследования сопоставляется с таблицей со столбцом дискриминатора, чтобы указать, какой конкретный тип в иерархии представлен в строке.</span><span class="sxs-lookup"><span data-stu-id="57059-821">Table per Hierarchy (TPH) – where each inheritance set maps to a table with a discriminator column to indicate which particular type in the hierarchy is being represented in the row.</span></span>
